# gitTest，纯粹是为了学习Git的操作和使用
学习git的使用

 - git init 初始化一个仓库
 - git clone 克隆远程仓库到本地
 - git add 当修改了文件之后，将修改的文件进行跟踪
 - git commit 将修改的文件添加到本地版本库
 - git checkout 切换分支
 - git pull 从远程仓库拉取master版本，更新到本地
 - git push 将本地版本更新到远程连接分支
 - git status 查看当前的状态
 - git branch 查看分支
 - git merge 合并分支
 - branch modified
